# Quiz Project

A simple, interactive quiz application built with HTML, CSS, and JavaScript. Users can answer multiple-choice questions and receive instant feedback on their performance.

## üéØ Project Brief

The goal of this project was to create an engaging and user-friendly quiz application that allows users to:
- Answer multiple-choice questions
- Track their progress through the quiz
- Receive instant scoring and feedback
- Reset and retake the quiz as many times as desired
- Access clear instructions on how to play

## üõ†Ô∏è Approach

### Technologies Used
- **HTML5** - Structure and semantic markup
- **CSS** - Styling, layout, and responsive design
- **Vanilla** - Quiz logic, DOM manipulation, and user interactions

### Development Strategy
The project was approached with a focus on:
1. **User Experience** - Creating an intuitive interface with clear instructions
2. **Accessibility** - Ensuring the quiz is easy to navigate and understand
3. **Responsiveness** - Making sure the application works across different devices
4. **Clean Code** - Writing maintainable and well-structured code

### Biggest Challenges
The anticipated challenges for this project included:
- **State Management** - Tracking user selections and maintaining quiz state without using a framework
- **DOM Manipulation** - Efficiently updating the UI based on user interactions
- **Score Calculation** - Accurately calculating and displaying results
- **User Flow** - Creating a smooth experience from start to finish, including modals and navigation
- **Responsive Design** - Ensuring the quiz looks good on all screen sizes

## üéÆ How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/deividsv95/quiz-project.git
   cd quiz-project
   ```

2. **Open the project**
   - Simply open `index.html` in your web browser
   - Or use a local server like Live Server in VS Code

3. **Take the quiz**
   - Click "How to Play" to read the instructions
   - Select your answers for each question
   - Click "Submit" to see your score
   - Use "Reset" to try again
